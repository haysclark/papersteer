public override function steerToAvoid(v:AbstractVehicle, minTimeToCollision:Number):Vector3

XXX 4-23-03: Temporary work around (see comment above)

Checks for intersection of the given spherical obstacle with a volume of "likely future vehicle positions": a cylinder along the current path, extending minTimeToCollision seconds along the forward axis from current position.

If they intersect, a collision is imminent and this function returns a steering force pointing laterally away from the obstacle's center.

Returns a zero vector if the obstacle is outside the cylinder
